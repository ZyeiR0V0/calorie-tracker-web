## 📖 关于项目 (About)

这是我在 GitHub 上的第一个开源项目。开发这个网站的初衷是帮助用户直观地记录每日饮食，自动计算热量（Calories）以及蛋白质、碳水化合物、脂肪等核心营养素的摄入情况。

通过这个项目，我学习并实践了全栈开发流程，包括前端交互设计、后端数据处理以及数据库的连接与管理。

## ✨ 主要功能 (Key Features)

* **🔍 食物搜索与录入**：支持搜索常见食物并添加到每日记录中（可接入第三方 API 如 USDA/OpenFoodFacts）。
* **📊 营养素可视化**：通过图表（Chart.js/ECharts）展示每日/每周的热量及三大营养素（蛋白质、碳水、脂肪）占比。
* **📅 历史记录查询**：回顾过去日期的饮食情况，追踪长期变化趋势。
* **📱 响应式设计**：完美适配桌面端与移动端，随时随地记录。
* **👤 用户系统**：支持用户注册、登录及个性化数据（如基础代谢率 BMR 计算）保存。
